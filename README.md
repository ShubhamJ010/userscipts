# userscipts
A collection of useful userscripts to enhance your web browsing experience.

## Available Userscripts

### Instagram Auto Reels Scroller

Automatically scroll to the next Instagram Reel when one ends. This userscript provides a seamless viewing experience on Instagram by automatically transitioning between reels without manual intervention.

**Features:**
- Automatically scrolls to the next reel when the current one ends
- Toggle functionality to turn the auto-scrolling on/off
- Visual status indicator showing whether the script is active
- Configurable settings via Tampermonkey
- Keyboard shortcut (Shift + S) to toggle the functionality
- Options to control scroll direction and number of plays before scrolling

**Installation:**
- Install a userscript manager like Tampermonkey
- Install the `instagram-auto-reels-scroller.user.js` script
- The script will run automatically on Instagram Reels pages

**Usage:**
- The script runs automatically when visiting Instagram Reels
- Use Shift+S keyboard shortcut or the Tampermonkey menu to toggle auto-scrolling
- A visual indicator in the top-right corner shows whether the auto-scrolling is on or off

### Quick Search

Quickly search for selected text on various platforms. This userscript allows you to highlight text on any webpage and instantly search for it on Google or YouTube using keyboard shortcuts.

**Features:**
- Highlight text and press keyboard shortcuts to search
- Search on Google (press 'g' or 'G')
- Search on YouTube (press 'y' or 'Y')
- Works on all websites (matches https://* and http://*)

**Installation:**
- Install a userscript manager like Tampermonkey
- Install the `Quick_Search.js` script
- The script will run automatically on all web pages

**Usage:**
- Select/highlight any text on a web page
- Press 'g' to search the selected text on Google
- Press 'y' to search the selected text on YouTube
